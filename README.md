# 🚗 Car Price Prediction using Linear Regression & Lasso Regression

A Machine Learning web application that predicts the selling price of a used car using Linear Regression and Lasso Regression models. The application is built with Python, Scikit-learn, and Streamlit.

---

## 🚀 Live Demo

https://car-price-prediction-linear-laapp-g6xnakhpszq9ppw7adxzfs.streamlit.app/
---

## 📌 Features

- Predicts used car prices
- Built using Linear Regression and Lasso Regression
- Interactive Streamlit web application
- Fast and accurate real-time predictions

---

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Streamlit
- NumPy
- Pandas
- Pickle

## 📊 Input Features

The model predicts the selling price based on features such as:

- Year
- Present Price
- Kms Driven
- Fuel Type
- Seller Type
- Transmission
- Owner

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/HarshitDataLab/car-price-prediction-linear-lasso.git
```

Move into the project folder:

```bash
cd car-price-prediction-linear-lasso
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
